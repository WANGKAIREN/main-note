# VSCode快捷键

## 选中
| 快捷键               | 解释                             |
| -------------------- | -------------------------------- |
| ctrl/cmd + d         | 依次找出文中所有的当前选中的单词 |
| ctrl/cmd + shift + l | 一次性找出文所有的当前选中的单词 |

## Markdown
| 快捷键          | 解释       |
| --------------- | ---------- |
| alr + shift + f | 格式化表格 |
